== Description ==

Looking for a tool for adding sticky content on your WordPress site pages/posts?

Try Conditional Content plugin for FREE - it will help you to insert and control the conditional content with a set of rules that you can configure (for example , Conditional Content allows you to easy add advertisement blocks, messages, warning, reminders, welcome/introduction texts, static text etc). The communication is achieved by handling WordPress functions.

== Installation ==

PHP 5.5 or higher is required!

To install the plugin:

1. Log in as administrator to WordPress Admin Panel.
2. Navigate to Plugins > Add New > Upload Plugin.
3. Browse for 'conditional-content'.zip file > Install Now.

== Uninstallation ==

To uninstall the plugin:

1. Log in as administrator to WordPress Admin Panel:
2. Navigate to Plugins > Installed Plugins > Conditional Content.
3. Press Deactivate > Delete. The plugin should be successfully deleted.

== Upgrade Notice ==

To update the plugin:

1. Log in as administrator to WordPress Admin Panel.
2. Navigate to Plugins > Installed Plugins > Conditional Content.
3. Press Deactivate button. Press Delete button. The plugin should be successfully uninstalled.
4. Navigate to Plugins > Add New > Upload Plugin.
5. Browse for 'conditional-content'.zip file > press Install Now button.
The plugin should be successfully re-installed.

== Configuration ==

To open plugin`s settings page:

1. Log in as administrator to WordPress Admin Panel: Settings > Conditional Content.
2. Add your rules into JSON section (or just import the settings). Save.
3. Check the results.

== Error Log ==

Errors are stored at $WP_HOME/wp-content/uploads/logs/
